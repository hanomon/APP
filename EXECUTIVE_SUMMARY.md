| 문제 | 연간 예상 손실 | 영향 |
|------|---------------|------|
| 📸 **QR 코드 캡처 및 공유** | 직원당 15-30일 허위 출근 | 급여 부정 지급 |

## 📱 주요 기능 (비기술적 설명)

### 직원용 모바일 앱
```
🔵 간단한 로그인 → QR 코드 자동 생성 → 태블릿에 보여주기 → 완료!
```
- 📱 스마트폰만 있으면 OK
- ⚡ 3초 만에 출퇴근 완료
- 📊 내 출퇴근 기록 실시간 확인
- 🎯 사용법이 너무 간단해서 교육 불필요

### 태블릿 QR 스캐너
```
📷 회사 입구에 태블릿 비치 → QR 코드 자동 스캔 → 출퇴근 처리 완료!
```
- 🖥️ 태블릿 1대면 충분
- 🔄 자동 스캔으로 터치 불필요
- ✅ 성공/실패 즉시 피드백
- 💰 별도 하드웨어 구매 불필요

### 관리자 대시보드
```
💼 실시간 현황 → 직원 관리 → 통계 확인 → 엑셀 다운로드!
```
- 📊 **실시간 출퇴근 현황**: 누가 출근했는지 한눈에
- 👥 **직원 관리**: 신입/퇴사자 등록/삭제
- 📈 **통계 분석**: 일별/주별/월별 리포트
- 📥 **엑셀 내보내기**: 급여 시스템 연동

### 자주 묻는 질문 (FAQ)

**Q: 도입 비용이 정말 50만원인가요?**
A: 네, 오픈소스이므로 라이선스 비용 없이 서버/태블릿 비용만 발생합니다.

**Q: 기존 ERP 시스템과 연동되나요?**
A: 네, RESTful API를 제공하여 대부분의 시스템과 연동 가능합니다.

**Q: 직원들이 사용하기 어렵지 않나요?**
A: 스마트폰으로 로그인 → QR 생성 → 보여주기 3단계로 매우 간단합니다.

**Q: 보안은 안전한가요?**
A: HMAC 암호화 + GPS 검증 + 30초 갱신으로 금융권 수준의 보안을 제공합니다.

**Q: 유지보수는 어떻게 하나요?**
A: 자체 IT팀이 관리하거나, 선택적으로 유지보수 계약 가능합니다.
